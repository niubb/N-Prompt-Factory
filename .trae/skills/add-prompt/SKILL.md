# 添加提示词技能

## 技能描述
本技能用于在提示词管理项目中添加新的提示词。

## 使用说明

### 执行步骤
1. 确定提示词名称（建议使用小写字母和连字符，如 `greeting-prompt`）
2. 编写提示词内容
3. 将提示词保存到 `prompts/` 文件夹下
4. 文件格式为 Markdown（.md）或纯文本（.txt）

### 文件命名规范
- 使用小写字母、数字和连字符
- 避免空格和特殊字符
- 文件扩展名：`.md`（推荐）或 `.txt`

### 输出格式

#### Markdown 格式
```markdown
# 提示词名称

提示词内容...
```

#### 纯文本格式
```
提示词内容...
```

## 示例

### 示例 1：创建问候提示词
文件名：`prompts/greeting.md`
```markdown
# 问候提示词

你好！我是一个AI助手，很高兴为您服务。
```

### 示例 2：创建写作提示词
文件名：`prompts/writing-guide.md`
```markdown
# 写作指导提示词

请帮我写一篇关于人工智能的文章，要求：
1. 介绍AI的基本概念
2. 讨论AI的应用场景
3. 分析AI的未来发展趋势
```

## 注意事项
1. 确保文件名唯一，避免覆盖现有提示词
2. 提示词内容应清晰、明确
3. 可以使用 Markdown 格式增强可读性
