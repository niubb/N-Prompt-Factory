## Why

目前项目的 skills 散落在各个子目录中，缺乏统一的入口页面。创建一个 HTML 索引页可以方便浏览和发现所有可用的 skills，提升开发者体验。

## What Changes

- 新增 `index.html` 作为 skills 索引页面
- 每个 skill 卡片展示名称、描述、操作按钮（查看/使用）
- 响应式布局，支持桌面和移动端
- 静态生成，无需后端

## Capabilities

### New Capabilities
- `skill-index-html`: 提供一个美观的 HTML 页面来索引和浏览所有可用的 skills
